**************************************
Deploy FindFace Security
**************************************


To make your user experience great, we offer 2 types of deployment. Choose the most convenient one, given your architecture outline:

* The cluster deployment can only be done step-by-step.
* The standalone deployment can be done step-by-step and from a developer-friendly console installer.
